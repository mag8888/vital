# Импорт товаров из инвойса - Прямо сейчас!

## Вариант 1: Через Railway CLI (рекомендуется)

```bash
cd /Users/ADMIN/Vital/vital
railway run npm run import-invoice-now
```

## Вариант 2: Через Railway Shell в браузере

1. Откройте https://railway.app
2. Перейдите в проект Vital
3. Откройте Shell/Terminal в последнем deployment
4. Выполните:

```bash
cd vital
npm run build
node scripts/import-invoice-now.js
```

## Вариант 3: Через админку (после миграции)

1. Откройте https://vital-production-82b0.up.railway.app/admin/invoice-import
2. Скопируйте содержимое файла `INVOICE_DATA.txt`
3. Вставьте в текстовое поле
4. Нажмите "Импортировать (синхронно)"

## Что будет импортировано:

- Все товары из файла `INVOICE_DATA.txt`
- Автоматический расчет продажных цен
- Обновление количества товаров
- Создание новых товаров, если их нет
- Уведомления о низком остатке (≤3 шт)
- Деактивация товаров с нулевым остатком

## Важно перед импортом:

1. ✅ Убедитесь, что миграция базы данных выполнена
2. ✅ Настройте курс и мультипликатор в админке (`/admin/invoice-settings`)
   - Курс: 2.45
   - Мультипликатор: 8
