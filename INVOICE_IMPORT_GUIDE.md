# Руководство по импорту инвойса

## Доступ к функциям

### Страницы:
- **Импорт инвойса**: `/admin/invoice-import`
- **Настройки импорта**: `/admin/invoice-settings`

### API Endpoints:
- `GET /admin/api/import-settings` - Получить текущие настройки
- `POST /admin/api/import-settings` - Сохранить настройки
- `POST /admin/api/import-invoice` - Импорт инвойса (фоновый режим)
- `POST /admin/api/import-invoice-sync` - Импорт инвойса (синхронный, с результатом)

## Настройка импорта

1. Перейдите на страницу **Настройки импорта** (`/admin/invoice-settings`)
2. Установите:
   - **Курс обмена**: 2.45 (БАТ → Рубль)
   - **Мультипликатор**: 8
3. Нажмите "Сохранить настройки"

**Формула расчета цены:**
```
Цена в БАТ × Мультипликатор × Курс = Цена в рублях
Цена в рублях / 100 = Цена в PZ
```

**Пример:**
- Закупочная цена: 453.86 БАТ
- Расчет: 453.86 × 8 × 2.45 = 8,895.66 руб
- В PZ: 8,895.66 / 100 = 88.96 PZ

## Импорт инвойса

1. Перейдите на страницу **Импорт инвойса** (`/admin/invoice-import`)
2. Подготовьте данные в формате:
   ```
   SKU|Описание|Количество|Цена в БАТ|Сумма
   ```
   
   Пример:
   ```
   FS1002-24|Rudis Oleum Botanical Face Care Night Formula 24 G|20|453.86|9077.20
   FS0001-24|Natural Balance Face Serum 24 G|6|348.72|2092.32
   ```

3. Вставьте данные в текстовое поле
4. Выберите режим импорта:
   - **Синхронный** - дождитесь результата на странице
   - **Фоновый** - результат будет отправлен в Telegram

## Что делает импорт

- ✅ Находит товары по SKU или создает новые
- ✅ Обновляет цены по формуле: `цена_закупки × 8 × 2.45 / 100`
- ✅ Устанавливает остатки из инвойса (не 999!)
- ✅ Переводит названия на русский язык (если настроен AI)
- ✅ Группирует товары с одинаковым SKU (суммирует количество)
- ✅ Деактивирует товары с нулевым остатком

## Формат данных

Каждая строка должна содержать 5 полей, разделенных символом `|`:

```
SKU|Описание|Количество|Цена в БАТ|Сумма
```

- **SKU** - код товара (например: FS1002-24)
- **Описание** - название товара
- **Количество** - количество единиц
- **Цена в БАТ** - цена за единицу в тайских батах
- **Сумма** - общая сумма (обычно Количество × Цена)

## Важно

- ⚠️ Товары с одинаковым SKU будут объединены (количество суммируется)
- ⚠️ Остатки берутся из инвойса, а не устанавливаются в 999
- ⚠️ Импорт может занять несколько минут из-за перевода названий
