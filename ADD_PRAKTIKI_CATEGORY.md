# Добавление категории "Практики"

## Вариант 1: Через скрипт (рекомендуется)

```bash
npm run add-praktiki-category
```

## Вариант 2: Через админку

1. Зайдите в админку: https://vital-production-82b0.up.railway.app/admin
2. Перейдите в раздел "Категории" (Categories)
3. Нажмите кнопку "Create new"
4. Заполните:
   - **Name**: Практики
   - **Description**: Категория для практик и методик (опционально)
   - **Is Active**: ✅ (включено)
5. Нажмите "Save"

## Вариант 3: Через админку (страница товаров)

1. Зайдите в админку: https://vital-production-82b0.up.railway.app/admin/products
2. При добавлении нового товара нажмите кнопку "+" рядом с полем "Категория"
3. В модальном окне заполните:
   - **Название категории**: Практики
   - **Описание категории**: Категория для практик и методик (опционально)
4. Нажмите "Создать категорию"

## Проверка

После добавления категории она должна появиться:
- В админке в списке категорий
- На главной странице веб-приложения при наличии товаров в этой категории
- В выпадающем списке при создании/редактировании товаров
