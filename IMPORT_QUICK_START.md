# ⚡ Быстрый старт: Импорт продуктов

## Шаги:

1. **Проверьте переменные окружения в Railway:**
   - ✅ `OPENAI_API_KEY` - для перевода
   - ✅ `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET` - для изображений
   - ✅ `DATABASE_URL` - для базы данных

2. **Запустите скрипт на Railway:**

   **Вариант 1: Через Railway CLI**
   ```bash
   railway run npm run import-siam
   ```

   **Вариант 2: Через SSH в контейнер**
   - Откройте Railway Dashboard
   - Выберите ваш сервис
   - Нажмите "Open Terminal"
   - Выполните:
   ```bash
   npm run build
   node dist/scripts/import-siam-botanicals.js
   ```

   **Вариант 3: Через деплой**
   - Скрипт можно запустить как часть деплоя
   - Или добавить как одноразовую команду

3. **Наблюдайте за логами:**
   - Скрипт выведет прогресс для каждого продукта
   - Покажет успешные и неудачные импорты
